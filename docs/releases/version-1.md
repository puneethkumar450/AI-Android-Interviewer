# Version 1 Release Scope

## Included

- Question Bank
  - Offline Room-backed storage
  - Search across questions and ideal answers
  - Bookmark support
  - Category and difficulty filters

- AI Mock Interview
  - Chat-style interview screen
  - Gemini prompt layer
  - Follow-up question generation
  - Instant feedback and provisional scoring
  - End-of-session evaluation

- Progress Tracking
  - Completed interview persistence
  - Category score summaries
  - Weak-area detection
  - Recommended study topics

## Deferred For Later Releases

- Dedicated interview practice drills
- Rich analytics dashboards
- Export/shareable reports
- Multi-module physical split into `core`, `data`, and `feature` modules
